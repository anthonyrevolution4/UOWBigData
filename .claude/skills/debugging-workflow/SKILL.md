# Debugging workflow — UOWBigData

> University big data project

## 1. Reproduce
- Capture exact error, command, and environment.
- Note: Python version, Node version, or relevant runtime.

## 2. Dependencies
- Install required packages per project config.
- Check for version conflicts.

## 3. Environment
- Verify required env vars are set.
- Check file paths and permissions.

## 4. Tests
- Run project-specific test suite.

## 5. Document
- Add test or note for the root cause once fixed.
